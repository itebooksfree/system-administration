Book Description
-----------------

The CompTIA A+ Quick Reference (220-701, 220-702) is a late-stage exam prep resource designed to be used as review shortly before your scheduled CompTIA A+ exam (2009 objectives). It is not intended to be a comprehensive curriculum. If you are using this resource, you should have already learned the material through a class or formal study method. The CompTIA A+ Quick Reference provides brief, straight forward, explanations on the major topics of the two CompTIA A+ exams â€” the 220-701 Essentials Exam and the 220-702 Practical Applications Exam.
