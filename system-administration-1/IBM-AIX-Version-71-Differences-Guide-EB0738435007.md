Book Description
-----------------

This IBM® Redbooks® publication focuses on the enhancements to IBM AIX® Version 7.1 Standard Edition. It is intended to help system administrators, developers, and users understand these enhancements and evaluate potential benefits in their own environments.
AIX Version 7.1 introduces many new features, including: - Domain Role Based Access Control - Workload Partition enhancements - Topas performance tool enhancements - Terabyte segment support - Cluster Aware AIX functionality
AIX Version 7.1 offers many other new enhancements, and you can explore them all in this publication. For clients who are not familiar with the enhancements of AIX through Version 5.3, a companion publication, AIX Version 6.1 Differences Guide, SG24-7559, is available.
