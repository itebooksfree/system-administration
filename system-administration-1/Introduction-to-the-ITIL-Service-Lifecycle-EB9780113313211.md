====================================================================================================================

Author : [AXELOS](https://ebookreading.net/search/author/AXELOS)

Release Date : 2011/12/01

Book Description
-----------------


    
    This guidance introduces IT service management and ITIL. It summarises the best practices described in the 2011 editions' core guidance, explaining the basic concepts of ITIL and providing information on each stage of the service lifecycle.
