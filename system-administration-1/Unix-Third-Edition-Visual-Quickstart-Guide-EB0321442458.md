Book Description
-----------------

Need to learn Unix fast? Try a Visual QuickStart! This best-selling reference’s visual format and step-by-step, task-based instructions will have you up and running with Unix software in no time. In this completely updated edition of our best-selling guide to Unix, Second Edition: Visual QuickStart Guide, leading software application experts Deborah S. Ray and Eric J. Ray use crystal-clear instructions and friendly prose to introduce you to Unix. Filled with step-by-step, task-based instructions and loads of visual aids, this book explains how to the most common Unix commands.
