Book Description
-----------------


    
    The Release, Control and Validation (RCV) syllabus, this quick-reference guide will help you as you study for the RCV module of the ITIL Intermediate Capability qualification. Beyond the exam, this handbook is a practical resource that can be used in the workplace.
