Book Description
-----------------

Cloud services are just as susceptible to network outages as any other platform. This concise book shows you how to prepare for potentially devastating interruptions by building your own resilient and reliable applications in the public cloud. Guided by engineers from 9apps, you’ll learn how to combine AWS with open source tools such as PostgreSQL, MongoDB, and Redis.
