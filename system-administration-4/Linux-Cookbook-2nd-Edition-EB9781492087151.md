Book Description
-----------------


    
    This handy cookbook teaches new-to-intermediate Linux users the essential skills necessary to manage a home or small business network. All of the recipes in this book are useful for any Linux system, including local area networks that involve iOS- or Android-powered devices. Youâ€™ll learn how to install, maintain, and troubleshoot a Linux system, add and remove software, manage filesystems, run backups, and more.
Carla Schroder, author of over a thousand Linux how-tos for various publications, teaches you the solid Linux foundations you need to build and run your network. How do you multiboot? Or troubleshoot software, hardware, and network issues? Each recipe addresses a specific problem and includes a discussion that explains the solution and provides insight into how it works.
